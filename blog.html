Discuss the scope of var, let, and const?

Answer:

Scope of variable refers to the accessibility of a particular variable within the program.

The scope of var:

  • The keyword var in javascript is used to declare a variable that is global and function scoped.
  • Var type variables can be redeclare and reassigned.
  • Var type variables are initialized with undefined.

The scope of let:

  • The keyword let in javascript is used to declare a variable that is block scoped.
  • Let type variables can be reassigned but can not be redeclare.
  • Let type variables are not initialized.

The scope of const:

  • The keyword const in javascript is used to declare a variable that is block scoped.
  • Const type variables can not be reassigned and redeclare.
  • Const type variables are not initialized.
